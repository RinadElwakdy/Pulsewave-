# PulseWave

PulseWave is a youth-focused landing page designed to support creativity, awareness, and personal growth.

## 🌟 Features
- Bilingual support (English / Arabic)
- Modern animated UI
- Volunteer & idea submission forms (frontend)
- Responsive design

## 🚀 How to Run
Just open `index.html` in any browser.

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
